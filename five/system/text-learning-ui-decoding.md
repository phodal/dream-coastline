# 文字学习与 UI 解码系统

## 核心概念

玩家刚进入墨颀时，和纪子轩一样看不懂这个世界。

随着学习墨颀文字，游戏 UI、NPC 对话、地图、道具说明会逐渐解码。

## 初始状态

NPC 对话示例：

> □□□！□□，□□！

任务目标示例：

> 任务目标：□□□

道具说明示例：

> 名称：？？？
> 用途：未知

## 学会基础字后的变化

学会“门”后，玩家可以看懂部分门牌、路标、机关提示。

学会“火”后，玩家可以识别能源设施。

学会“名”后，玩家可以看到部分 NPC 名字。

学会“约”后，玩家可以阅读契约和法律文本。

## 示例进化

初始：

> 【？？】说：“□□□，□□，□□！”

学会“门”“名”后：

> 【守门人】说：“外乡人，门，不开！”

进一步学习后：

> 【守门人】说：“外乡人，没有名册记录，不得入内。”

## 学习方式

### 课堂学习

在闻素或续文院处学习基础字。

### 环境学习

通过观察路牌、符文、机关，推断字义。

### 战斗学习

某些字必须在危险中第一次真正掌握。

### 任务学习

完成相关支线后解锁字根。

### 文明学习

后期通过标准字典和国书网络批量解锁高级组合。

## 游戏效果

文字学习不仅是剧情设定，也影响：

- UI 可读性。
- 地图探索。
- 战斗技能。
- NPC 信任。
- 支线选择。
- 城市建设。
- 结局路线。

## 设计意义

这个系统能让玩家直接体验“从文盲到执笔者”的成长，而不是只在剧情里听角色说自己在学习。
