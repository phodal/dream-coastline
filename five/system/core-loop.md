# 核心循环

## 游戏总体循环

探索世界 → 学习文字 → 解决问题 → 建设文明 → 解锁新区域/新技术 → 推进主线。

## 前期循环

1. 进入陌生区域。
2. 遇到看不懂的文字或规则。
3. 通过 NPC、书院、任务学习新字。
4. 用新字解决谜题或战斗。
5. 获得更多理解与地图权限。

体验重点：玩家和纪子轩一起从无知到理解。

## 中期循环

1. 接到城市或地区问题。
2. 分析问题涉及的文字、工程、社会矛盾。
3. 选择解决方式：教育、工程、政治、探索。
4. 改变城市指标。
5. 解锁新文明技术。

体验重点：玩家不是单人冒险，而是在推动文明转型。

## 后期循环

1. 选择文明发展路线。
2. 建设大型工程。
3. 通过续页推进时间。
4. 看到前期选择的长期后果。
5. 集结资源完成归星计划。

体验重点：玩家看到一个文明成长起来。

## 终章循环

1. 现代区域静默化。
2. 玩家建立临时国书节点。
3. 恢复名字、灯光、地图、记忆。
4. 对抗熄星者静默。
5. 完成最终 Continue。

体验重点：两个世界的系统合流。

## 玩家长期目标

表面目标：找到父母，回家。

中期目标：让墨颀活下去。

后期目标：让墨颀拥有观测星空与跨界救援能力。

终局目标：阻止现代星球被静默。
