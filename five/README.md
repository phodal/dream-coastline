## 目录说明

- `project/`：项目总览、阅读顺序、七幕大纲和叙事原则。
- `world/`：墨颀文明、现代星球、文字现实系统和反派协议。
- `people/`：主角、夏离、父母、配角与反派势力。
- `scene/`：按序幕到第七幕拆分的游戏场景与章节。
- `system/`：核心循环、文字学习、战斗、建设、存档和章节模板。
- `script/`：序幕与第一章脚本样稿。
- `ending/`：结局路线。
- `appendix/`：术语表和后续待定问题。
