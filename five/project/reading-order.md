# 墨颀游戏企划资料包

这是一个按文件拆分的游戏企划草案包，基于当前设定整理：

- 现代少年纪子轩
- 古代文字文明“墨颀”
- 夏离王子
- 从零学习墨颀文字
- 墨颀从王朝发展为文明
- 最终墨颀反过来拯救纪子轩所在星球
- “To be Continue ... / Continue”作为核心主题与系统

## 推荐阅读顺序

1. `01_项目总览.md`
2. `02_故事总大纲_七幕版.md`
3. `世界观/01_墨颀文明.md`
4. `角色/纪子轩.md`、`角色/夏离.md`
5. `剧情分幕/00_序幕_灯未亮起的夜晚.md`
6. `剧情分幕/01_第一幕_不会写字的人.md`
7. `系统设计/02_文字学习与UI解码.md`
8. `系统设计/03_组字战斗系统.md`
9. `系统设计/04_文明建设系统.md`
10. `脚本样稿/第一章开场脚本.md`

## 文件用途

- `剧情分幕/`：适合继续扩写成主线任务。
- `角色/`：适合继续扩写人物弧光、台词、支线。
- `世界观/`：适合统一设定，避免后续写崩。
- `系统设计/`：适合转成游戏机制文档。
- `脚本样稿/`：适合继续写成实际游戏剧本。
- `附录/`：放术语表、待定问题、后续开发清单。

## 当前定位

类型建议：剧情 RPG + 文字解谜 + 文明养成 + 章节式冒险。

核心卖点：

> 一个现代少年被古代文明救走，又陪它长大，直到这个古代文明反过来救回他的世界。
