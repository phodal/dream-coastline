## 单章阅读重点

- `场景定位`：这一章在整部游戏里的职责。
- `玩家流程`：玩家实际经历的关卡顺序。
- `关键节点`：必须落地的剧情转折。
- `玩法接口`：需要系统支持的交互、战斗或建设机制。
- `角色推进`：主要角色在本章结束时发生了什么变化。
- `交接到下一幕`：下一章为什么自然发生。

旧版 `1.md` 是最早的第一幕草稿，已拆分吸收到序幕和第一幕场景卡中。
