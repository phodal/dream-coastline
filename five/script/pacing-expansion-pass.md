# 剧本扩写 Pass：时长与体感

## 目标

当前 playable JSON 的估算时长约 3 小时，但很多段落仍像七幕梗概：玩家看到了大事件，却没有足够时间经历失败、停顿、误会、修正和角色关系变化。

这一轮补强不追求把所有章节写成长篇，只先解决最影响体感的四类缺口：

1. 第二幕的学习过程太顺，需要失败和复盘。
2. 第三幕的夏离转折太快，需要更多证据压到他身上。
3. 第五幕的百年跨度太大，需要代际场景承载时间。
4. 第七幕的最终章信息太密，需要现代日常被静默的细节和最终回应。

## 第二幕：墨颀书院

### 场景 A：闻素的基线测试

目的：让“不会写字”不是一句设定，而是一次被闻素拆开的学习诊断。

纪子轩以为闻素会像游戏导师一样直接教字。闻素先给他一张纸，上面混着日常文字、律文字和真文字。

闻素：

> 不要猜。把你看不懂的地方圈出来。

纪子轩圈了整张纸。

夏离皱眉。

> 你连这些都看不懂？

闻素没有笑。

> 这很好。至少他知道自己不知道。

这个段落对应 playable JSON 的 `baseline` 调查点，目的是让学习从“拿技能”变成“承认无知 -> 建立基线 -> 学习规则”。

### 场景 B：第一次修井失败

目的：让文字工程有代价。玩家不能抄对字形就成功。

纪子轩照着旧符抄写“止”，井沿裂纹短暂停住，随即向外多裂出一圈。

夏离上前想补救。

闻素拦住他。

> 让他看完。

水声消失，排队的孩子安静下来。

纪子轩：

> 我写错了？

闻素：

> 你描对了形，没理解它为什么要停在这里。

这个失败不是惩罚玩家，而是让“复盘”成为第四幕续文院的前置情绪。

### 场景 C：字典页边批注

目的：把知识垄断落到物证上。

地下藏书室的基础字典边缘有多层批注：第一层是教学笔记，第二层是安全警告，第三层是墨律司删除线。

纪子轩读不全，只能看懂被划掉的几个名字。

夏离：

> 这些字不是没人学过。

闻素：

> 是学过，又被收回了。

### 场景 D：母亲的节点批注

目的：把父母线从“影像揭露”提前改成“工程痕迹”。

第一个国书节点修复后，影像出现前，纪子轩看到母亲留下的一行小字：

> 不要只修这个节点。要让每个人都知道它为什么会坏。

这句话把第二幕、第四幕和第五幕连成一条线：修复不是终点，公开修复能力才是终点。

## 第三幕：死去的王国

### 场景 A：没有买主的集市

目的：死城不能只靠废墟表达，要有“秩序仍在、生活已死”的反差。

摊位每天自动开张，价格牌自动更新，巡逻符器按时经过。但没有人讨价还价，没有孩子偷看糖，连争吵都没有。

纪子轩：

> 这里看起来不像毁了。

夏离：

> 所以它更像死了。

### 场景 B：学生退回的信

目的：证明夏离不是只想复国，他确实推动过开放教育。

藏书楼里有一叠没送出去的信：

> 殿下，我们已经会写自己的名字了。

每封信末尾都盖着墨律司的红印：退回。

纪子轩把信递给夏离。夏离没有接，只看着信封上的名字。

### 场景 C：审讯抄本

目的：让墨律司的反派性从“追捕者”变成制度性恐惧。

审讯抄本没有问夏离是否叛国，只反复问：

> 谁允许你把字教给无名之人？

纪子轩这时才明白，墨律司真正害怕的不是夏离，而是普通人开始拥有定义自己的能力。

### 场景 D：烧黑的教学墙

目的：让夏离承认自己的错误不是口头转折，而是看见代价。

宫城倒塌长廊里有半面烧黑的墙，上面还留着儿童练字。

夏离：

> 我以为他们只需要机会。

纪子轩：

> 他们还需要时间。

夏离：

> 我没有给他们时间。

### 场景 E：夏离的回答

目的：Boss 战前后完成“复国王子 -> 文明重建者”的实际转折。

国书残影问：

> 无王之国，何以为国？

夏离一开始想回答“我会成为王”。但他看见学生信、审讯抄本和烧黑教学墙后改口：

> 若我只是让王座重新坐满，这座城还是会死第二次。

## 第五幕：百年续页

### 场景 A：闻素的空椅

目的：用缺席表达时间，而不是直接说“很多年过去”。

讲堂前排保留一把旧椅子。新教师点名时，学生们会向那把椅子点头。

纪子轩问：

> 她还会来吗？

学生回答：

> 她来过。现在轮到我们继续教。

### 场景 B：夜校灯光

目的：让文明成长不只体现在高塔，也体现在普通人日常。

工人下工后坐进夜校。他们手上有矿灰，写字很慢。有人把“名”写错，旁边的人没有嘲笑，而是把错误记到复盘板上。

这回失败不再是纪子轩一个人的事，而是公共学习流程的一部分。

### 场景 C：夏离逐渐国书化

目的：让夏离的代价可感知。

纪子轩在两个城市相隔很远的节点里都看见夏离的投影。夏离仍然会开玩笑，但语气有延迟。

纪子轩：

> 你现在到底在哪里？

夏离：

> 在墨颀需要我在的地方。

停顿后他又说：

> 这不是坏事。只是我有时会忘记自己原本坐在哪一把椅子上。

### 场景 D：第一次跨界失败

目的：星象工程不能一次成功。失败要让现代危机提前漏进来。

跨界信标第一次只收到一段噪声。纪子轩听见像是学校铃声，下一秒整段记录被擦白。

星象工程师没有说“失败”。他们把数据贴上墙，开始争论失败原因。

纪子轩第一次意识到：回家需要的不是奇迹，而是一个不会被失败击垮的文明。

## 第七幕：灯重新亮起

### 场景 A：父母留下的回声

目的：回家后不要立刻解释父母真相，先让玩家听见残缺指令。

录音只剩断句：

> 如果你回来了，不要先找我们，先让这座城重新被看见。

纪子轩站在同一张餐桌前。序幕里他只想找父母，这一次他必须先救城市。

### 场景 B：缺席的班主任

目的：把静默落到个人记忆，而不是只做灾难背景。

学校点名表上，班主任那一栏是空的。

纪子轩记得她总在拖堂，却想不起她的声音。

夏离：

> 这就是失名？

纪子轩：

> 不。更糟。像我本来就不该记得。

### 场景 C：恢复姓名的小票

目的：让“呼名”任务有一个很小但可回收的现代证据。

店员恢复名字后，收银小票重新打印出她的姓名。她盯着那几个字，确认自己还在这个世界。

纪子轩没有说大道理，只把小票递给她。

> 收好。等会儿如果你又忘了，就看这个。

### 场景 D：失败的归桥测试

目的：最终章也要失败一次，避免一路铺到 Boss。

父母实验室第一次归桥同步失败，墙面和桌子短暂变成空白。

夏离稳住主国书。

纪子轩把错误写进方案。

父母记录中传来母亲的声音：

> 很好。不要删掉失败记录。

### 场景 E：文明回声

目的：最终战的胜利不能像主角单挑，要让前面建设出的文明一起回应。

最终协议删除队友名后，墨颀各城同时回写：

> 该文明选择继续。

这句不是口号。它来自学塾、档案馆、工坊、星象塔、现代信标和临时国书节点的共同备份。

玩家最后写下“名，门，星，归，续”时，屏幕上不应该只有纪子轩和 Boss，而应该显示两个文明的系统都在参与。

## 验收

这一轮完成后，必须满足：

- `data/story_scenes/*.json` 的 `min_minutes` 不再全是 5 分钟。
- Godot `--smoke-autoplay` 仍然通过所有八幕。
- `tools/ascii_five.py <scene> --verify` 可以正确执行含 `engage` 的 walkthrough。
- 新增调查点不是 filler：每个都对应失败、证据、代价、关系或后续回收。
